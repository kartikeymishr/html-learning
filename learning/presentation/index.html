This webpage aims to show the properties of HTML Images

The alt property of the < img > tag replaces the image with an alternate text chosen by the developer in the case the image fails to load i.e., when the browser cannot locate the image that the markup is referencing to.

The < figure > element in HTML is used to markup images in a document. It specifies self-contained content. One notable child-element of the < figure > element is the < figcaption > element that is used to give a suitable caption to the image in question, as seen below.
